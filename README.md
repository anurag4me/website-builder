# Website Builder

An AI-powered website builder that generates and previews websites in real-time using WebContainer.

## Project Structure

- [Frontend/](Frontend/) - React frontend application
- [Backend/](Backend/) - Express.js backend service

## Getting Started

1. Set up the backend:

```
cd Backend
npm install
```

Create .env with your OpenAI API key

```
npm run dev
```

2. Set up the frontend:

```
cd Frontend
npm install
npm run dev
```

3. Open http://localhost:5173 in your browser

## Features

- AI-powered website generation
- Real-time preview using WebContainer
- Interactive code editor
- Step-by-step building process
- Support for React and Node.js projects

## Tech Stack
Frontend: React, TypeScript, Tailwind CSS, WebContainer
Backend: Express.js, OpenAI API
